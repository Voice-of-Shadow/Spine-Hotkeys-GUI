# Spine Hotkeys Editor

**简体中文** | [English](README.en_US.md)

</div>

---

Spine Hotkeys Editor 是一个用于修改Spine快捷方式的界面工具，基于 Pyside6 开发。旨在替代Spine 非常原始的快捷键修改方式。


# 功能特性

---

- **可视化快捷键编辑**
- **快捷键冲突检测**
- **集成两种录入模式**
- **添加多语言支持**
- **路径链接与临时备份**

# 使用说明

---

#### 核心使用指引

1. **链接快捷键文件**：首次启动时。点击"链接"按钮、选择Spine快捷键文件。`hotkeys-1.txt (v4.0+)`
2. **选择录入模式**：通过点击"单引号模式"激活单引号模式或切换回一般输入模式。
3. **编辑快捷键**：选中条目进行编辑。
4. **查看说明**：点击右侧对应的按钮可以查看使用说明、快捷键说明、关键字字符对应、程序信息。
5. **保存更改与重载**：点击主窗口的"确认"后、重启Spine或在Spine编辑器设置中重新加载快捷键。

#### 其他操作说明

- **备份**：点击"打开文件夹"按钮打开链接文件的路径、对快捷键文件进行备份。
- **恢复修改前的快捷键**：将Spine的快捷键文件替换为`./processing/`目录下的快捷键文件。
- **恢复默认快捷键**：删除快捷键文件或在Spine编辑器设置中点击恢复默认。
- **多语言切换**：点击右方下拉框选择对应的语言。需要确保存在对应的语言包且在配置中启用。


# 快捷键说明

---

- 一般模式可以使用"ctrl"、\"shift"、"alt"修饰键组合快捷键。
- 单引号模式(字符模式)仅可以使用"alt"修饰键组合快捷键。
- 使用单引号模式、快捷键将附加长按功能。在触发动作一段延迟后重复触发。
- 单引号模式可以录入单个可键入字符('a'、'ア'、'啊'、'↓')。部分字符有对应关键字。
- 单引号模式键盘外特殊字符的键入需要做键映射、请不要做超预期的录入。


# 适配说明

---

GUI程序通过解析Spine快捷键文件结构、进行数据转换。所以功能理论上是覆盖目前所有版本的Spine快捷键文件。

显示的文本会使用语言包中对应的值。由于不同Spine版本的快捷键有些不同，所以有些快捷键键名没有记录在语言包中，此时会显示原本的名称。

因为快捷键基本是随版本更新而增加，所以目前提供的语言包是使用Spine最新版本的快捷键文件制作的。

- 特别说明：本人不是专业软件开发人员。代码、配置和程序逻辑设计等可能存在不合理不标准不规范的情况。功能添加、代码完善、项目优化将交给后来者了(当然还是希望官方能写一个。坑都挖了几年了都)。
- 特别说明：本人不会英语。默认的英文文档使用的是机器翻译。存在英文本地化优化的需求。
- 特别说明：此程序仅适配标准美式键盘。
- 特别说明：当前最新测试版下(4.3.39-beta)。单引号模式下、'PLUS' → 会被Spine编辑器提示为无效字符、'+' 会直接使报错并导致所有热键无法使用。虽然设计没禁止输入。但是请展示这个字符当成超预期输入不要使用。已经得到官方的反馈表示将在下一个版本中得到修正。
- 特别说明：当前程序预处理阶段无法排除所有超预期的快捷键(即格式合法、但键不合法或无法被Spine编辑器识别)。
